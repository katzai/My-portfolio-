1 Use the Enhanced Admin Panel

Simply use admin-enhanced.html instead of admin.html

You can either:

  • Rename admin-enhanced.html to admin.html
  • Or access it directly at admin-enhanced.html

‚úì Default PIN: 1234 (change in the file if needed)

2 Add Auto-Sliding to Your Portfolio

Open your index.html and add this script tag before the closing </body> tag:

<!-- Enhanced Animations and Auto-Sliding --> <script src="js/enhancements.js"></script> </body>

Make sure the js folder exists with enhancements.js inside it.

3 Update Project Loading Function

Find your displayProjects() or loadProjects() function in index.html and add this code at the end:

function displayProjects() { // ... your existing project display code ... // ADD THIS AT THE END: setTimeout(() => { if (typeof initProjectSliders === 'function') { initProjectSliders(); } if (typeof initAutoScrollProjects === 'function') { initAutoScrollProjects(); } }, 500); }
4 Test Everything

Checklist to verify everything works:

  • Open admin-enhanced.html and login (PIN: 1234)
  • Create a new folder
  • Upload multiple images
  • Try cropping an image
  • Create a project with multiple images
  • View your portfolio (index.html)
  • Watch project images auto-slide
  • Watch project cards auto-scroll
  • Hover over cards to see enhanced animations
